# LLM Prompts: Kolejność Użycia

W tym CMS-ie używaj **2 promptów w tej kolejności**:

1. [docs/llm-prompt-01-environment.md](C:/xampp/htdocs/ercoding-cms/docs/llm-prompt-01-environment.md)
2. [docs/llm-prompt-02-project.md](C:/xampp/htdocs/ercoding-cms/docs/llm-prompt-02-project.md)

## Jak tego używać

- Prompt 1 uruchamiasz na początku projektu.
- Prompt 1 ma ogarnąć środowisko XAMPP, bazę danych, reset startera i zostawić czysty szkielet CMS.
- Dopiero po wykonaniu promptu 1 wklejasz prompt 2.
- Prompt 2 służy do właściwego kodowania strony, sekcji, stylów i integracji treści z CMS.

## Cel

Po promptcie 1 ma zostać:

- działające środowisko lokalne na XAMPP,
- świeża baza danych,
- działający panel i funkcje CMS,
- wyczyszczone treści projektu, maile, telefony, linki, uploady i stare assety projektu,
- sam szkielet Ercoding CMS z wytycznymi dla LLM.

Po promptcie 2 ma powstać:

- kompletna, edytowalna strona zbudowana zgodnie z zasadami CMS,
- proste, bezpieczne i przewidywalne rozwiązania,
- kod czytelny dla juniora, ale technicznie solidny.
